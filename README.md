# Trip-Planner
 Smart Travel Planning System

## Overview
Trip Planner helps users build day-wise itineraries, recommends places, and estimates travel budgets.

## Features
- Destination input + user preferences
- Place recommendations
- Day-by-day itinerary
- Budget estimation (transport, stay, food)
- Save and load plans

## Tech Stack
- Python 3.x
- Flask (or FastAPI) for backend
- SQLite / JSON for storage
- HTML/CSS/JS (optional frontend)

## Setup (local)
1. Clone: git clone https://github.com/yourusername/trip-planner.git
2. Create venv: python3 -m venv venv && source venv/bin/activate
3. Install: pip install -r requirements.txt
4. Run: python src/main.py
